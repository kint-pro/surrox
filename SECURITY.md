# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in surrox, please report it responsibly.

**Do not open a public issue.**

Email **mikyta.dev@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce (proof of concept)
- Potential impact

We will acknowledge your report within 5 business days and work with you to understand and address the issue before any public disclosure.
